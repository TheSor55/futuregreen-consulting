# AQL Sampling Dashboard

## Project Overview

AQL Sampling Dashboard is a QC tool for calculating inspection sample size, code letter, acceptance number, rejection number, and Accept / Reject decisions for factory quality inspection.

The project helps quality inspectors and factory teams make sampling decisions more consistently using AQL logic.

## Business Problem

Quality inspection teams often rely on printed tables, manual lookup, or individual experience when applying AQL sampling. This can lead to inconsistent decisions, slower inspection work, and unclear records for audit or customer review.

A simple digital dashboard helps inspectors select the right sampling plan and communicate results more clearly.

## Objective

- Support practical AQL sampling decisions.
- Reduce manual lookup work for inspectors.
- Provide a simple dashboard for QC training and demonstration.
- Help factories standardize inspection decisions before shipment or supplier acceptance.

## Target Users

- QC inspectors
- QA / QC managers
- Incoming quality control teams
- Final inspection teams
- Supplier quality teams
- Factory audit preparation teams
- Consultants supporting quality-system improvement

## Key Features

- AQL level selection
- Inspection level selection
- Lot size and sample size calculation
- Code letter lookup
- Acceptance and rejection number calculation
- Real-time Accept / Reject decision
- Industry presets and recommended inspection settings
- Inspection history for basic trend review
- CSV / PDF export support
- Bilingual Thai / English interface
- Offline-capable single-page web app

## Project Status

Prototype / MVP

The tool is suitable for demonstration, training, and early workflow use. It is not a validated enterprise QC system and does not include user accounts, controlled master data, database storage, approval workflow, or production audit controls.
